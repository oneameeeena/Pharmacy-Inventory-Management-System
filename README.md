# Pharmacy-Inventory-Management-System

🏥 Pharmacy Inventory Management System

A Python console application designed to help manage a pharmacy’s inventory and prescription system efficiently. This project demonstrates object-oriented programming concepts and provides a menu-driven interface for easy testing of all features.

🔹 Features

Medicine Categories: Manage different types of products such as Medicine, Supplements, Personal Care, and Medical Devices.

Medicine Management: Add, remove, display, and search medicines by name or description.

Prescription Management: Create and manage customer prescriptions, including adding/removing medicines, calculating total costs, and canceling prescriptions.

Sales Tracking: Calculate total sales amount from all active prescriptions.

Interactive Menu: Simple and intuitive menu system to test all functionalities.

🔹 Technologies Used

Python 3

Object-Oriented Programming (OOP)

Lists & Dictionaries for data management

datetime module for tracking prescription dates
